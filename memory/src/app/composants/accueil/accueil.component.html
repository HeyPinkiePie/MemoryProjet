Bienvenue sur le site du Memory.
Venez jouer avec nous et faites le meilleur score possible.

Règles du jeu :

  • Retournez deux cartes. Si les images sont identiques, le jeu laisse les deux images visibles.
  • Si les images diffèrent, les cartes se retournent face cachée là où elles étaient.
  • La partie est terminée lorsque toutes les cartes ont été assemblées par paires.

Inscrivez-vous via le formulaire pour jouer et sauvegarder vos préférences de grilles.
Si vous voulez relancer une partie, cliquez sur la barre d'espace.
